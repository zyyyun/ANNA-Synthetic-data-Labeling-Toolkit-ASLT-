## Out of Scope (final — milestone 종료 시점)

| Feature | Reason |
|---------|--------|
| 새 기능 추가 (배치 처리, 키보드 설정 등) | GS인증은 기존 기능 완성도 평가, 새 기능은 추가 결함 리스크 |
| 크로스 플랫폼 지원 | WinForms는 Windows 전용 |
| MVVM 아키텍처 전환 | 대규모 리팩토링 = 기능 변경 리스크 |
| 상용 라이선스 시스템 전면 교체 | MAC 기반 인증 개선만 |
| 단위 테스트 프레임워크 도입 | 인증에서 테스트 코드 자체 미평가 |
| 4K 60fps 지원 | 새 기능, WinForms 구조로 불가능 |
| 다국어(영어) 지원 | 인증 범위 밖 |
