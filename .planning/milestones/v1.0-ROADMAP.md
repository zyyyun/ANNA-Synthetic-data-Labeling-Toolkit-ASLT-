## Overview

GS인증 1등급(ISO/IEC 25023 8대 품질 특성) 통과를 위한 품질 개선 마일스톤. 기존 구현된 기능을 결함 없이 동작하도록 개선하되 새 기능은 추가하지 않는다. 로그 인프라를 먼저 구축하고(Phase 1), 안정성·기능 정확성·성능·설치 환경 순으로 개선한 뒤(Phase 2~5), 잠긴 바이너리에 대한 사용자 보고 결함을 결함수정 phase 로 일괄 처리(Phase 5.6)했다. 문서화(Phase 6)는 본 마일스톤 범위에서 제외 — Known Gaps 로 이월.

## Milestone Summary

**Decimal Phases:**
- Phase 5.5: 기능 보정 + 안정화 (Phase 5 후 inserted, Waypoint 탐색 + 영상 로드 가드)
- Phase 5.6: 결함수정 (Phase 5 후 inserted 2026-04-23, KTC + QA 결함 17건)

**Key Decisions:**
- 기존 기능만 개선, 새 기능 추가 안 함 (GS인증은 기존 기능 완성도 평가)
- 대규모 아키텍처 리팩토링 제외 (기능 변경 리스크 최소화)
- KISA 가이드 기반 보안 강화 (SHA-256 + Salt PBKDF2-HMAC, 310,000 iterations)
- Static LogService class + Serilog daily file rotation + [AUDIT] prefix
- HmacChainSink (ILogEventSink) + HmacKeyProvider (LocalAppData/.hmac-key 32B RNG) + LogIntegrityVerifier — HMAC-SHA256 체인 무결성 검증, 30일 보존, 감사 이벤트 13종
- Cross-day chain continuity — 프로세스 내 rollover 는 in-memory _lastHmac 유지 / 재시작 시 디스크 recovery / 첫 boot 에 prior-day 파일 fallback
- ChangeBoxIdOnly: Waypoint.ObjectId 자동 변경 없이 선택 박스만 ID 변경 (NEW-05 locked)
- ResolveIdForNewBox 5단계 ID 승계 (Waypoint→entryFrameIndex→selectedBox→currentAssignedId→기본1)
- Self-contained publish + Inno Setup 6 + AppId pinned for in-place 1.0.x upgrades
- 로그 파일명 ASLT-yyyy-MM-dd.log (v1.0.2 에서 AOLT 잔재 정리)

**Issues Resolved (highlights):**
- Vehicle 드롭다운 선택 교체 버그
- JSON 타임스탬프 TimeSpan 기반 정확화
- 6개 경로 bbox 좌표 클램핑
- 영상 전환 시 상태 완전 초기화
- Generic exception → 구체적 예외 + 한국어 메시지
- Undo/Redo: AddBox/RemoveBox/ModifyBox + Waypoint 삭제 atomic Undo (v1.0.2)
- Cold-decoder seek 폭주 차단 (RELI-06: _isVideoReady ready-gate)
